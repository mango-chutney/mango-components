Tristicon

You will need to include the tristicon fonts with webpack copy or a font loader (or w/e you please).

Another option is to use the Base 64 encoded WOFF file inline, so you dont have to copy font files. You can use the createTristiconFontFace export for this:


injectGlobal`
  ${createTristiconFontFace()}
`;

Example


<Tristicon icon="facebook" />
